# 개발 계획서

## 추가 개발 항목

### 1. Hugging Face 경량 LLM 통합
- Hugging Face Hub API를 통한 모델 브라우징
- 모델 다운로드 및 로컬 저장
- 모델 메타데이터 관리 (크기, 태그, 다운로드 수 등)
- 경량 모델 필터링 (7B 이하, 양자화 모델 우선)

### 2. 로컬 모델 서빙 환경
- Ollama 기반 모델 서빙
- 모델 자동 시작/중지
- 서빙 상태 모니터링
- 여러 모델 동시 서빙 지원

### 3. 사용자 AI 프롬프트 화면
- 대화형 채팅 인터페이스
- 프롬프트 히스토리 관리
- 컨텍스트 기반 프롬프트 제안
- RAG 통합 프롬프트 (문서 참조)

### 4. PostgreSQL 마이그레이션
- SQLite → PostgreSQL 전환
- 데이터베이스 연결 설정
- 마이그레이션 스크립트
- 상호 복제 설정 가이드

### 5. 벡터 임베딩 및 RAG 기능 확인
- ✅ 벡터 임베딩: sentence-transformers 사용
- ✅ RAG 검색: FAISS 벡터 DB 사용
- ✅ 문서 인덱싱 및 검색 기능

## 구현 순서

1. PostgreSQL 설정 및 마이그레이션
2. Hugging Face 통합 (모델 브라우징/다운로드)
3. 로컬 모델 서빙 환경
4. 사용자 AI 프롬프트 화면
5. 통합 테스트
