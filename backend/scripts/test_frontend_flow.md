# 프론트엔드 플로우 테스트 가이드

## 테스트 전 준비

1. 백엔드 서버 실행
```bash
cd backend
uvicorn app.main:app --reload
```

2. 프론트엔드 서버 실행
```bash
cd frontend
npm start
```

3. 테스트 데이터 생성 (선택사항)
```bash
cd backend
python scripts/create_test_data.py
```

## 예상 동작

### 정상 동작
- 모든 페이지가 정상적으로 로드됨
- API 호출이 성공적으로 완료됨
- 데이터가 화면에 정확히 표시됨
- 오류 메시지가 적절히 표시됨

### 오류 처리
- 네트워크 오류 시 적절한 메시지 표시
- 권한 오류 시 접근 제한 메시지
- 데이터 없음 시 빈 상태 메시지
